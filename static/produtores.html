Consultar Produtores

Conforme a API oficial Applyfy: GET https://app.applyfy.com.br/api/v1/gateway/admin/producers. Autenticação no servidor com x-public-key e x-secret-key (nunca expostas ao browser). O painel usa o proxy /api/gateway/producers.

Query suportados: page, pageSize (5–50, padrão 20), nameOrEmail, phone, status, kycStatus, bankDataStatus, minDocumentsSent, accountType, tags (IDs separados por vírgula). Vários status aceitam valores separados por vírgula.
